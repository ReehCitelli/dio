## 💡 Aprendizados

- Estrutura e conceitos de bancos de dados relacionais
- Modelagem com MER e DER
- Criação e manipulação de tabelas com SQL
- Operações CRUD — INSERT, SELECT, UPDATE, DELETE
- Normalização de dados — 1FN, 2FN e 3FN
- Consultas com JOINs para combinar múltiplas tabelas
- Subconsultas para filtros e cálculos complexos
- Funções agregadas e agrupamento de resultados
- Índices para otimização de performance
