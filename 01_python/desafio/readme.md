# 🧠 Desafios de Código — Lógica e Pensamento Computacional

Pasta com os desafios de código do módulo **Aperfeiçoe Sua Lógica e Pensamento Computacional** do Bootcamp TOTVS — Fundamentos de Engenharia de Dados e Machine Learning.

Os desafios seguem a metodologia **PBL (Problem Based Learning)** — aprender resolvendo problemas reais, focando em lógica e pensamento computacional antes de partir para o código.

---

## 📁 Estrutura

```
desafios/
├── desafio_01_desconto_pedido.py
├── desafio_02_aprovacao_pedido.py
└── README.md
```

---

## 📋 Desafios

| # | Arquivo | Descrição | Conceitos |
|---|---------|-----------|-----------|
| 01 | [desafio_01_desconto_pedido.py](./desafio_01_desconto_pedido.py) | Calcula o valor final de um pedido após aplicar um percentual de desconto | `input`, `float`, `int`, operadores aritméticos, f-string |
| 02 | [desafio_02_aprovacao_pedido.py](./desafio_02_aprovacao_pedido.py) | Decide se um pedido deve ser aprovado, revisado ou rejeitado com base em valor e prioridade | `input`, `split`, `int`, `if/elif/else`, operador `in` |

---

## 💡 Aprendizados

- Leitura e conversão de dados de entrada com `input()`, `split()`, `int()` e `float()`
- Desempacotamento de variáveis em uma única linha
- Estruturas condicionais com `if`, `elif` e `else`
- Uso do operador `in` para verificar valores em listas
- Formatação de saída com f-strings e `:.2f`
- Identificação de inconsistências entre enunciado e exemplos — habilidade essencial no mundo real

---

## 🚀 Como executar

```bash
# Entre na pasta
cd dio/python/desafios

# Execute passando a entrada manualmente
python desafio_01_desconto_pedido.py
# Digite: 150.00 10
# Saída:  135.00

python desafio_02_aprovacao_pedido.py
# Digite: 800 alta
# Saída:  aprovado
```
